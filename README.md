## Setup
Clone
```
git clone https://github.com/davidwrenner/Qiskit-Gate-Art.git
```
(optional) Create & Activate Virtual Environment
```
virtualenv env

source env/bin/activate
```
Install Requirements
```
pip install -r requirements.txt
```
Usage
```
./generate.py text
```
